# Conversion Calculator for Android
### Description
This application is used to complete unit conversions. Allows for the following conversions:
- Length
  - Meters -> Yards
  - Meters -> Miles
  - Yards -> Meters
  - Yards -> Miles
  - Miles -> Yards
  - Miles -> Meters
- Volume
  - Liters -> Gallons
  - Liters -> Quarts
  - Gallons -> Liters
  - Gallons -> Quarts
  - Quarts -> Liters
  - Quarts -> Gallons
